# 🛍️ Catálogo de Loja com Painel Administrativo

Projeto de um sistema completo de catálogo de produtos com painel administrativo, carrinho, avaliações; desenvolvido com auxílio de IA utilizando a plataforma Lovable.

🔗 **Acesse o site:**
https://orioncoldres.lovable.app/

---

## 📌 Sobre o Projeto

Este projeto foi criado para atender a necessidade real de uma loja, permitindo que o próprio dono gerencie seus produtos de forma simples, sem precisar de conhecimentos técnicos.

A aplicação oferece uma interface intuitiva para administração do catálogo, com suporte a login seguro, upload de imagens e persistência de dados em nuvem.

---

## 🚀 Funcionalidades

* 🔐 Autenticação de usuário (login)
* 👤 Controle de acesso com perfil **admin**
* 📦 Cadastro de produtos
* ✏️ Edição de produtos existentes
* ❌ Remoção de produtos
* 🖼️ Upload de imagens diretamente do dispositivo
* ☁️ Persistência de dados em nuvem
* 📱 Interface responsiva e fácil de usar

Bônus: Seção de Avaliações do Cliente, Carrinho de Compras que encaminha direto para o whatsapp do dono da loja;

------

## 🧠 Objetivo

Criar uma solução prática para pequenos negócios gerenciarem seus produtos online, com foco em:

* Facilidade de uso (UX simples para usuários leigos)
* Automação de tarefas administrativas
* Estrutura pronta para evolução futura

---

## 🏗️ Tecnologias Utilizadas

* Frontend moderno (gerado via IA)
* Backend integrado (database + autenticação)
* Armazenamento de imagens em nuvem
* Integração com serviços gerenciados

> Projeto desenvolvido com auxílio de IA utilizando a plataforma Lovable

---

## 🔐 Acesso Administrativo

O sistema possui um painel administrativo protegido por login, onde é possível:

* Adicionar novos produtos
* Atualizar informações
* Gerenciar imagens
* Remover itens do catálogo
