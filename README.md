## The Project
  
Complete web-based system for managing product catalogs, with an administrative panel, shopping cart, and review area.

## Demo

Visit the website: https://orioncoldres.lovable.app/

---

## About the Project

The goal of this project is to offer a practical solution for small businesses that need to manage their products online in a simple and intuitive way, without depending on advanced technical knowledge.

The application features an administrative panel protected by authentication, allowing complete catalog management, including product registration, information editing, image upload, and cloud data persistence.

---

## Features

* User authentication
* Administrative access control
* Product registration
* Product editing and updating
* Product removal
* Image upload
* Cloud data persistence
* Responsive interface
* Shopping cart
* Order forwarding via WhatsApp
* Customer review system

---

## Project Objectives

* Facilitate product management for small businesses
* Reduce manual administrative processes
* Provide a simple experience for non-technical users
* Provide a scalable base for future evolutions

---

## Technologies and Structure

* Modern frontend developed with AI assistance
* Backend integrated with authentication and database
* Cloud image storage
* Integration with managed services

---

## Administrative Panel

The system has an administrative area protected by login, where it is possible to:

* Add new products
* Update catalog information
* Manage images
* Remove Products

---

## Key Features

* Complete application with integrated frontend and backend
* User-focused interface
* Structure prepared for expansion

## Sobre o Projeto
