# Catálogo de Loja com Painel Administrativo

Sistema web completo para gerenciamento de catálogo de produtos, com painel administrativo, carrinho de compras e área de avaliações.

## Demonstração

Acesse o site: https://orioncoldres.lovable.app/

---

## Sobre o Projeto

O objetivo deste projeto é oferecer uma solução prática para pequenos negócios que precisam gerenciar seus produtos online de forma simples e intuitiva, sem depender de conhecimentos técnicos avançados.

A aplicação conta com um painel administrativo protegido por autenticação, permitindo o gerenciamento completo do catálogo, incluindo cadastro de produtos, edição de informações, upload de imagens e persistência de dados em nuvem.

---

## Funcionalidades

* Autenticação de usuários
* Controle de acesso administrativo
* Cadastro de produtos
* Edição e atualização de produtos
* Remoção de produtos
* Upload de imagens
* Persistência de dados em nuvem
* Interface responsiva
* Carrinho de compras
* Encaminhamento de pedidos via WhatsApp
* Sistema de avaliações de clientes

## Painel Administrativo

O sistema possui uma área administrativa protegida por login, onde é possível:

* Adicionar novos produtos
* Atualizar informações do catálogo
* Gerenciar imagens
* Remover produtos
